Project description

This is my first project which was created as a thesis at the end of my Python Developer course at TeachMeSkills IT-school. I developed this project on my own. The mentor offered to choose a project to implement (a Telegram bot, a blog, etc.). My choice was an online-store, although, the mentor said that itâ€œs better to implement such projects in a team. But I decided to do it alone.

My choice was in favour of an online-store because the full implementation of this project will require the use of various technologies, libraries and the Django framework itself, which will be a good experience for me.

The result is a correctly working website for an online store with the following functionality:

  • Creating a microservice architecture with Docker-Compose: containers with application, database (PostgreSQL), proxy server (Nginx);
  • Pretty nice and responsive UI;
  • User registrations+captcha, private office
  • Catalogue with product categories, subcategories
  • Description of each product, product photo, current price, quantity available; for authorized users: the ability to add to cart/favorites, write feedback and rate the product
  • Display of product prices in US dollars and euros at the current NB RB exchange rate;
  • Display of exchange rates for major currencies on each page;
  • Promo code system. When a promotional code is applied, the user is informed of the amount of the discount;
  • Payment system (PayPal-sandbox version);
  • Email newsletter with store news;
  • nformation about current weather (OpenWeatherAPI) in user's city (for authorized users) or Minsk (for unauthorized users). When implementing this functionality, I gained experience in writing my own Middleware and context processor to get data about user's city before rendering a page template;
  • Contact form. When you fill in the form, an email is sent with the question and the customer's contact details for feedback;
  • Pages with information about the store, delivery, promotions and discounts;
  • Possibility to search for products by name;
  • Links to social media;
  • Docker containerize.
