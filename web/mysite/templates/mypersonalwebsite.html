Project description

The purpose of this project is to create a personal CV site with extended information about me and detailed descriptions of projects that are inappropriate to list on a resume

This is a small project which entirely performs the tasks assigned to it, so the functionality is quite limited. For the implementation of this project I chose Aiohttp-framework to work with the new framework, different from Flask, Django.

  • The UI was made on a Bootstrap 5 template with customization of individual elements (background, project-pages and others);
  • The application is containerized using Docker for easy deployment of the project;
  • Nginx in Docker-container with Docker-Compose capabilities was used as a proxy;
  • The project itself is located on a cloud configuration server Ubuntu 20.04 LTS.
