My path in IT started with clumsy attempts to create a site on Wix.com for my friend's organization. This area interested me and I decided to continue studying web development. I chose the backend, in particular Python, because this language is flexible, has extensive libraries, and a large number of application areas.

In 2021 I started my studies at TeacmMeSkills IT-school. During the training, I mastered the basic use of the Python language, its main libraries, got acquainted with the Django and Flask frameworks. I independently implemented an online-store in Django with the main functions of this kind of project as a graduation project.

I am convinced that the main thing for a modern person is to constantly learn and move forward. Today I am improving my skills in the backend, but I also try to get new knowledge from the frontend and DevOps area. In the future, I want to learn JavaScript, React.
