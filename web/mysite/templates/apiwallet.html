API Wallet project description The small project, which was written as a test task. The prerequisites was to create the project using DjangoRestFramework. It was necessary to implement registration system, basic CRUD, daily expense/income statistics distribution system, as well as application containerization, creation of Nginx proxy server of minimal microservice architecture using Docker-Compose.
