Project description

API Wallet project description This is a small API-application for controlling income and expenses.

Technologies:

  • DjangoRestFramework
  • Nginx
  • Docker/Docker-Compose
  • Djoser
  • JWT-authentication
  • PostgreSQL
  • SwaggerUI
  • Redis
  • Celery/Celery-Beat

It was decided to create the project using DjangoRestFramework. It was necessary to implement registration system, basic CRUD, daily expense/income statistics distribution system for each user, application containerization, creation Nginx proxy-server, creation of project documentation using Swagger.

Application functionality includes:

  • User registration, verification with email;
  • Authentication using JWT-tokens;
  • Initial initiation of a balance. Later it can be changed manually;
  • Income and expenditure operations (add, edit, delete operations);
  • Daily distribution of statistics for the previous day to all users;
  • Detailed documentation of the application;
  • Containerization of the whole project.
