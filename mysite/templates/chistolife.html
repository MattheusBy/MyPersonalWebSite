Project description

Главная цель данного проекта — создание полностью готового проекта для организации, включающее весь необходимый функционал для конкретной области, в частности химчистка мягкой мебели. Мой приятель занимался химчисткой мебели и он выступил в качестве гипотетического заказчика. Мы с ним обсудили какие именно функции сайта он хотел бы видеть для себя, что в последствие было реализовано. Основными задачей было создать сайт, при посещении которого клиент может самостоятельно получить максимум интересующей его информации без обращения к менеджерам.

Мной было принято решение использовать фреймворк Flask, так он позволяет реализовывать обширный круг опций, при этом данный фреймворк не обладает перегруженностью инструментами из коробки. Подключение БД, а так же система регистрации пользователей я счел неактуальным для данного приложения. Было решено реализовать следующую структуру сайта: • Главная страница содержит баннер с названием компании, примеры работ до и после чистки, раздел о преимуществах компании; • Страница с актуальными ценами для каждого вида мебели, а так же цены на дополнительные услуги; • Страница, описывающая процесс и этапы чистки; • Страница с онлайн-календарем с расписанием чисток по датам и времени. Необходимость продиктована тем, чтобы клиент заблаговременно мог узнать и выбрать удобную дату и время для него дату и время из представленных. Заполнение календаря производится менеджером при подтверждении клиентом заказа. • Страница, содержащая контактную информацию и ссылки на социальные сети. • Страница, позволяющая клиенту самостоятельно заполнить форму с необходимой информацией, а так же обязательным прикреплением фото мебели для точной оценки стоимости услуг, так как окончательная стоимость чистки не всегда соответствует цене в прайсе и зависит от степени загрязнения, конструктивных особенностей изделия и др. Данная опция позволяет клиенту не тратить время на выбор удобного для него мессенджера для связи с менеджером, а отправить информацию сразу с сайта. Сообщение приходит по средствам Telegram к менеджеру и после оценки клиент по SMS получает окончательную цену за чистку его изделия. • Реализован индивидуальный дизайн для страниц с ошибками 404 и 500. Дополнительной задачей для меня было создание адаптивного и приятного интерфейса. Для решения этой задачи был использован Bootstrap 5 с применением возможностей Canva, bootsnipp.com, svgwave.in, icons8.com. Для тестирования приложения были использованы фреймворк Pytest, инструмент Coverage, SeleniumWebDriver. Покрытие приложения тестами составляет 98%. В проекте применены возможности Docker и Docker Compose для упрощения процесса реализации проекта на сервере.
